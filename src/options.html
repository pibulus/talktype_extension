Note: For this demo, the actual transcription is simulated, so any text will work as an API key.

About this demo: In a production version, this extension would use a service like OpenAI's Whisper for transcription. For demonstration purposes, we simulate the transcription process.

Microphone Permission

The extension requires microphone access to record audio for transcription.

Checking permission status...

Important: For Chrome security reasons, extensions cannot automatically set microphone permissions to "Allow". Please follow these steps:

  1. Click "Open Chrome Microphone Settings" above
  2. Find the entry for this extension (chrome-extension://[extension-id])
  3. Change the setting from "Ask" or "Block" to "Allow"
  4. Return to this page and click "Test Microphone Access"

For Mac users: You must also allow Chrome in System Preferences > Security & Privacy > Privacy > Microphone

How to use:

  1. Enter your Gemini API key above and save.
  2. Grant microphone access using the button above.
  3. Navigate to any website with text inputs.
  4. Click on the microphone icon that appears next to text inputs.
  5. Speak your text and it will be transcribed into the input field.
