How to Use TalkType
  1. Enter your API key above and click Save Settings
  2. Grant microphone access by clicking Test Microphone
  3. Visit any website with text inputs
  4. Click the microphone icon that appears next to text inputs
  5. Start speaking and your words will be converted to text
